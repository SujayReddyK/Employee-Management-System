# Employee Management System

## Overview
Employee Management System is a Java desktop application developed to manage employee records efficiently. It allows users to perform CRUD (Create, Read, Update, Delete) operations using a MySQL database.

## Features
- Add Employee
- View Employee Details
- Update Employee Information
- Delete Employee
- MySQL Database Integration
- Java Swing GUI

## Technologies Used
- Java
- Swing
- JDBC
- MySQL
- IntelliJ IDEA
- Git & GitHub
